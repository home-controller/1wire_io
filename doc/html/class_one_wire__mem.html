int OneWire_mem::set_1wire_state_on_bus ( byte  state,
byte  chip,
bool  skipSaved 
)

Set the 1wire chip IO pin state.

if chip = 0 set all found with OneWire.search(addrArray) else call this many times and set that one. chip No. is the order found when using OneWire.search(addrArray). well one bus at a time. For more info see https://www.arduino.cc/reference/en/libraries/onewire/ or data-sheet for the 1-wire devises.

For return last chip found. If chip = 0 return total number of chips. else if chip > 0 return chip(the param passed in value) if it was found.)

Parameters
statepin state bitmap
chip1-wire chip number to find on all pins/bus.
skipSavedSkip chips saved to eeprom. //
saveTempSave the chip found in the tempSaved1wireChip var. Only 1 chip stored.
Returns
int return -1 if chip not found else Number of last chip found.

Set 1-wire chip IO pin bitmask(register on chip) for chip stored in EEProm. todo only handles DS2413.

Parameters
chipchip is the index for the array in EEPROM
channelchannel is a bit mask of the IO pins on the 1-wire chip. Set bits to 1 for the IO pins effected by the action.
actionaction = 0: off, 1: on, 2: toggle.
Returns
int8_t return -1 for invalid 'action' -2 for unhandled 1-wire chip type else return success of wiring to chip
